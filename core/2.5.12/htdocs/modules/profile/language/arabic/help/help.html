المساعدة: الملف الشخصي

الوصف

وحدة الملفات الشخصية مخصصة لإدارة حقول الملفات الشخصية المخصصة للمستخدمين.

التثبيت/الإزالة

لا توجد إجراءات خاصة ضرورية، اتبع عملية التثبيت القياسية - استخرج مجلد /profile إلى مجلد ../modules. قم بتثبيت الوحدة من خلال الإدارة -> وحدة النظام -> الوحدات.

تتوفر تعليمات مفصلة حول تثبيت الوحدات في دليل عمليات زووبس

تعليمات التشغيل

لإعداد هذه الوحدة، تحتاج إلى:

  • تكوين تفضيلاتك للوحدة (انظر "التفضيلات") واختياريًا مربع الملف الشخصي إذا كنت تنوي استخدامه (انظر "المربعات").
  • تعديل الفئات الموجودة أو إضافة فئات جديدة.
  • تعديل الحقول الموجودة أو إضافة حقول جديدة. هنا يمكنك تحديد الحقول التي ستكون مرئية في أي فئة، وما إذا كانت ستكون مرئية أثناء تسجيل المستخدم.
  • تحديد ترتيب خطوات التسجيل.
  • وأخيراً، يمكنك تعيين أذونات للحقول الفردية - أي منها قابل للتعديل، وأي منها قابل للبحث.

إجراءات مكافحة البريد العشوائي

لتقليل عمليات تسجيل البريد العشوائي، قم بما يلي:

  • انتقل إلى وحدة الحماية في الإدارة، وانتقل إلى التفضيلات، ثم في الأسفل، عند هذا الخيار: "إيقاف منتدى الرسائل غير المرغوب فيها" يتحقق من بيانات POST مقابل مرسلي الرسائل غير المرغوب فيها المسجلين في قاعدة بيانات www.stopforumspam.com.
    قم بتعيينه على "حظر عنوان IP (بلا حدود)"

  • في /class/captcha/config.php، تأكد من تعيين الوضع على "text":

    return $config = array(
    'disabled' => false, // تعطيل CAPTCHA
    'mode' => 'text', // الوضع الافتراضي، يمكنك اختيار 'text' أو 'image' أو 'recaptcha' (يتطلب مفتاح API)
    'name' => 'xoopscaptcha', // اسم التحقق
    'skipmember' => true, // تخطي فحص CAPTCHA للأعضاء
    'maxattempts' => 10, // الحد الأقصى للمحاولات لكل جلسة
    );

  • في وحدة الملف الشخصي، انتقل إلى الإدارة وفي الخطوة الأساسية الموجودة في:
    /modules/profile/admin/step.php?id=1
    قم بتعيين "حفظ بعد الإيقاف" إلى "لا"


  • في تفضيلات الملف الشخصي، قم بتعيين خيار "استخدام التحقق المرئي بعد الخطوة الثانية من التسجيل" إلى "نعم" (وهو الإعداد الافتراضي)
